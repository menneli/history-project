# HistorySounds
FastAPI веб-приложение, связывающее музыку XX века с историческими событиями. Данные импортируются из Excel, хранятся в SQLite и отображаются через Jinja2 шаблоны с полной поддержкой кириллических URL.

## Требования
- Python 3.10+
  ```

## Быстрый старт
1. **Установите зависимости**
   ```bash
   pip install -r requirements.txt  # или выполните команду выше
   ```

2. **Импортируйте и свяжите данные**
   ```bash
   python main.py
   ```
   *Создаёт `data/hs.db`, импортирует ~35 валидных композиций и 221 событие, связывая их через колонку `ID события`.*

3. **Запустите веб-сервер**
   ```bash
   uvicorn app:app --reload
   ```
   Откройте: http://localhost:8000 или 127.0.0.1 - в зависимости от вывода после исполнения команды

## Чек-лист проверки
| Тест | Ожидаемый результат |
|------|---------------------|
| **Таймлайн загружается** | Показывает 35 композиций, без `nan`, `?` или пустых записей |
| **Клик по композиции** | Переход на `/composer/song` или `/song/name` без 404 |
| **Кириллические URL** | Корректно обрабатывает названия вроде `«День Победы»` (браузер сам кодирует/декодирует) |
| **События отображаются** | У связанных композиций появляется секция "Связанные исторические события" с описаниями |
| **Неизвестные композиторы** | Композиции без композитора маршрутизируются на `/song/{name}` и показывают `(Unknown)` |
| **Нет проблем с кэшем** | Жёсткое обновление (`Ctrl+Shift+R`) показывает актуальный контент; CSS загружается полностью |

## Решение проблем
| Проблема | Решение |
|----------|---------|
| `jinja2 must be installed` | Выполните `python -m pip install jinja2` (используйте `python -m pip` для правильного окружения) |
| `304 Not Modified` / CSS не загружается | Жёсткое обновление (`Ctrl+Shift+R`) или откройте в режиме инкогнито |
| `NameError: song_event` | Убедитесь, что `models/connection.py` импортирован в обоих `Song` и `Event` |
| `nan` или пустые строки в таймлайне | Запустите `python main.py` снова; импортёр теперь фильтрует пустые строки Excel |
| Порт 8000 занят | Измените на `--port 8080` в команде uvicorn |

---
*Создано с помощью FastAPI, SQLAlchemy, Jinja2 и pandas. Пайплайн данных автоматически обрабатывает Excel edge cases (NaNs, кавычки, float ID, пустые строки).*
